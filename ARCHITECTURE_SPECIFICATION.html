Technical Specification Summary

The architecture follows a standard 3-tier decoupling pattern. The Frontend is built for high-performance React 19 rendering, communicating with a FastAPI backend via structured JSON REST protocols. Data integrity is enforced at the database level with SQLAlchemy 2.0 typed ORM and manual CTE recursive guards.
